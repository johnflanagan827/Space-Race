goal of game:
Space Race is a singleplayer game where the user controls a spaceship.
The goal of the game is to reach the end of the screen without hitting any debris.
If the spaceship hits debris, it is reset to the start position.
If the spaceship reaches the end of the screen, the score increases and the spaceship is reset to the start position.
The user must aim at getting the highest score before running out of time!

controls:
w: move spaceship up
s: move spaceship down
q: quit game
